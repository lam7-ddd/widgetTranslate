Widget Translate デモページ

このページは翻訳ウィジェットのテスト用デモページです。 右上の翻訳ボタンから言語を選択して、リアルタイム翻訳をお試しください。

主な機能

  • リアルタイムテキスト翻訳
  • 多言語サポート（英語、中国語、韓国語など）
  • 自動DOM監視による動的コンテンツ対応
  • 翻訳結果のキャッシュ機能
  • レスポンシブデザイン対応

技術仕様

このウィジェットは以下の技術を使用して構築されています：

  • バックエンド: Node.js + Express
  • 翻訳API: Google Cloud Translation API
  • フロントエンド: Vanilla JavaScript
  • スタイル: CSS3（ダークモード対応）

使用方法

ウェブサイトに以下のコードを埋め込むだけで、翻訳機能を追加できます：

<script src="https://your-domain.com/widget.js"></script>
